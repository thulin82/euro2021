# [EURO 2021](https://github.com/thulin82/euro2021)

## Setup

### Get project

```
$ git clone git@github.com:thulin82/euro2021.git
```

### Get dependencies

```
$ npm install
```

### Run app

```
$ npm start
```

### Environment variables
In config/config.env add
```
API_KEY =
API_KEY2 =
```
